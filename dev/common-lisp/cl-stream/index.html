CL-STREAM

An attempt to replace Common Lisp streams with streams supporting any data type and non-blocking semantics.

1 Features

Re-usable stream classes for streams, input streams, output streams, buffered streams, queue.

2 Contributing

https://github.com/cl-stream/cl-stream

3 Projects

Projects supporting or relying on cl-stream.

3.1 babel-stream

Charset encoding/decoding classes for cl-stream using Babel.

https://github.com/cl-stream/babel-stream

3.2 token-stream

Lexer classes for cl-stream.

https://github.com/cl-stream/token-stream

3.3 unistd-stream

A cl-stream stream using cffi-posix unistd semantics.

https://github.com/cl-stream/unistd-stream

4 Proposed direction

HTML documentation for stream classes.
