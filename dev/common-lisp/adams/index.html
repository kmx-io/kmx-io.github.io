Adams

UNIX system administrator written in Common Lisp.

1 Features

Adams produces commands for the shell (/bin/sh) for local or remote hosts using SSH in order to retrieve and modify a UNIX system status.

Currently it allows automated administration of users, groups and packages on Linux and OpenBSD without any additional requirement or installation on the target machines.
